# Consumer Insights Analytics

## End-to-End Customer Analytics Project using Python, Statistics, Machine Learning and Power BI

---

## Project Overview

Consumer Insights Analytics is an end-to-end analytics project that demonstrates how customer demographic and survey data can be transformed into actionable business insights.

The project covers the complete analytics lifecycle including:

- Data Preparation
- Exploratory Data Analysis (EDA)
- Descriptive Statistics
- Inferential Statistics
- Regression Analysis
- Customer Segmentation
- Decision Tree Classification
- Bootstrap Analysis
- Interactive Power BI Dashboard
- Business Recommendations

The objective is to help organizations understand customer behavior, identify key drivers of customer satisfaction, segment customers for targeted marketing, and support data-driven business decisions.

---

## Business Problem

Organizations collect customer demographic and survey data but often struggle to convert it into actionable insights.

This project answers key business questions such as:

- What factors influence customer satisfaction?
- Which customer segments are most valuable?
- How does product quality affect customer experience?
- Which customers require targeted marketing?
- How can business performance be improved using analytics?
