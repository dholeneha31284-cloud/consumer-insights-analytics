# Consumer Insights Analytics

## Overview

Consumer Insights Analytics is an end-to-end analytics project designed to analyze customer survey data, evaluate brand performance, measure customer satisfaction, and assess marketing campaign effectiveness.

The project demonstrates how survey data can be transformed into actionable business insights using SQL, Python, statistical analysis, and interactive dashboards.

---

## Business Objective

The primary objective is to help business stakeholders understand customer behavior, brand perception, and campaign performance to support data-driven decision-making.
